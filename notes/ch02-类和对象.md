# 第二章：类和对象

## 2.1 创建类

```java
package cn.comicjava.ch02;

public class DoSomething {
    
    public void sayHello() {
        System.out.println("Hello from DoSomething!");
    }
    
    public static void main(String[] args) {
        DoSomething instance = new DoSomething();
        instance.sayHello();
    }
}
```

- **类定义**: `public class 类名`
- **创建对象**: `new 类名()`
- **调用方法**: `对象.方法名()`
