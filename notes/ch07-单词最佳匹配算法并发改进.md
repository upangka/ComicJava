
# 单词最佳匹配算法 

基本是使用【】算法，在字典[UK Advanced Cryptics Dictionary.txt](../data/UK%20Advanced%20Cryptics%20Dictionary.txt)中找到与目标词，相似度最近的单词。




# 使用callable改进

上一个版本主要是在一个线程遍历字典中所有的单词，现在用分段的思想来处理。通过将字段分段给不同的线程进行处理，从而提升并发度。等所有的线程处理结束后再聚合结果。

定义任务要提交给线程池

```java
/**
 * 任务
 * @author 鲨鱼不喝Jvaa 抖音号:77283340926
 * @version 1.0
 * @since 2026/4/23
 */
public class BasicTask implements Callable<BestMatchingData> {
    private final int startIndex;
    private final int endIndex;
    private final List<String> dictionary;
    private final String targetWord;
    public BasicTask(int startIndex, int endIndex, List<String> dictionary, String targetWord) {
        this.startIndex = startIndex;
        this.endIndex = endIndex;
        this.dictionary = dictionary;
        this.targetWord = targetWord;
    }

    @Override
    public BestMatchingData call() throws Exception {
        List<String> words = new ArrayList<>();
        int minDistance = Integer.MAX_VALUE;
        for (int i = startIndex; i < endIndex; i++) {
            String str = dictionary.get(i);
            int distance = LevenshteinDistance.instance.minDistance(targetWord, str);

            if(distance < minDistance){
                words.clear();
                words.add(str);
                minDistance = distance;
            }else if(minDistance == distance){
                words.add(str);
            }
        }
        return new BestMatchingData(minDistance,words);
    }
}
```

> 核心代码

提交**任务** 给线程池，获取结果

```java
List<Future<BestMatchingData>> results = new ArrayList<>();
for (int i = 0; i < numCores; i++) {
    // ...
    // 异步提交
    Future<BestMatchingData> future = executor.submit(new BasicTask(startIndex, endIndex, dictionary, word));
    results.add(future);
}
```

聚合所有结果

```java
// 合并结果
List<String> words = new ArrayList<>();
int distance = Integer.MAX_VALUE;
for (Future<BestMatchingData> future : results) {
    try {
        BestMatchingData data = future.get();
        if(data.distance() < distance){
            words.clear();
            words.addAll(data.words());
            distance = data.distance();
        }else if(data.distance() == distance){
            words.addAll(data.words());
        }
    } catch (Exception e) {
        throw new RuntimeException(e);
    }
}
return new BestMatchingData(distance, words);
```
