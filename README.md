## 📁 项目结构

```
ComicJava/
├── src/                      # 源代码目录
│   └── com/
│       └── comicjava/        # 按包名组织
│           ├── Main.java     # 主入口
│           ├── ch01/         # 第一章
│           ├── ch02/         # 第二章
│           └── ...
├── bin/                      # 编译输出目录 (无需提交)
└── README.md
```

## 🛠️ 编译运行

```bash
# 编译 (在项目根目录)
javac -d bin src/com/comicjava/Main.java

# 运行
java -cp bin com.comicjava.Main
```
